## 配置使用

neovim 最好使用最新版，如果操作系统是 Ubuntu，请添加 neovim 最新版的 ppa

```bash
git clone https://github.com/miaohn/nvim.git ~/.config/nvim
```

在控制台输入`nvim`后，输入如下命令

```vim
:PackerSync
```
